## Pricing Overview

Tavily uses a credit-based billing model:

- **Free**: 1,000 credits/month
- **Pay-as-you-go**: $0.008 per credit (allows you to be charged per credit once your plan's credit limit is reached).
- **Monthly plans**: $0.0075 - $0.005 per credit
- **Enterprise**: Custom pricing and volume

| Plan | Credits per month | Monthly price | Price per credit |
|------|-------------------|---------------|------------------|
| **Researcher** | 1,000 | Free | - |
| **Project** | 4,000 | $30 | $0.0075 |
| **Bootstrap** | 15,000 | $100 | $0.0067 |
| **Startup** | 38,000 | $220 | $0.0058 |
| **Growth** | 100,000 | $500 | $0.005 |
| **Pay as you go** | Per usage | $0.008 / Credit | $0.008 |
| **Enterprise** | Custom | Custom | Custom |

Go to [billing](https://app.tavily.com/billing) to review plan options and manage your subscription.

## API Credits Costs

### Tavily Search

Your [search depth](/api-reference/endpoint/search#body-search-depth) determines request cost.

- **Basic Search (`basic`):**
  Each request costs **1 API credit**.

- **Advanced Search (`advanced`):**
  Each request costs **2 API credits**.

### Tavily Extract

The number of successful URL extractions and your [extraction depth](/api-reference/endpoint/extract#body-extract-depth) determine request cost. Failed URL extractions are not charged.

- **Basic Extract (`basic`):**
  Every 5 successful URL extractions cost **1 API credit**

- **Advanced Extract (`advanced`):**
  Every 5 successful URL extractions cost **2 API credits**

### Tavily Map

The number of pages mapped and whether natural-language [instructions](/api-reference/endpoint/map#instructions) are specified determine request cost. Failed map requests are not charged.

- **Regular Mapping:**
  Every 10 successful pages returned cost **1 API credit**

- **Map with (`instructions`):**
  Every 10 successful pages returned cost **2 API credits**

### Tavily Crawl

Tavily Crawl combines mapping and extraction, so request cost is the sum of both:

- **Crawl Cost = Mapping Cost + Extraction Cost**

For example:

- If you crawl 10 pages with basic extraction depth, you'll be charged **1 credit for mapping** (10 pages) plus **2 credits for extraction** (10 successful extractions divided by 5) which equals **3 total credits**
- If you crawl 10 pages with advanced extraction depth, you'll be charged **1 credit for mapping** plus **4 credits for extraction** which equals **5 total credits**

### Unlocking Production API Keys

<Steps>

<Step title="Select a paid monthly plan" icon="credit-card">
  
  Any paid plan unlocks production API keys and removes test-only restrictions.

</Step>

<Step title="Or enable pay-as-you-go" icon="settings">
  
  If you prefer not to commit to a plan, you can enable usage-based billing to unlock production API keys.

</Step>

</Steps>

### How Overage Billing Works

<Expandable title="What happens when I exceed my plan's credits?" default-open="false">

  
  When you use all credits included in your plan:

  - If **pay-as-you-go is enabled**, additional API calls continue and are billed at $0.008 per credit.
  - If **pay-as-you-go is disabled**, API requests may fail once you exceed your plan quota. Enable usage-based billing to avoid interruptions.

</Expandable>
