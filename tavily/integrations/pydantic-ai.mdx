## Introduction

Integrate [Tavily with Pydantic AI](https://ai.pydantic.dev/common-tools/#tavily-search-tool) to enhance your AI agents with powerful web search capabilities. Pydantic AI provides a framework for building AI agents with tools, making it easy to incorporate real-time web search and data extraction into your applications.

## Step-by-Step Integration Guide

<Steps>
  <Step title="Install Required Packages" icon="download">
    Install the necessary Python packages:

```bash
pip install "pydantic-ai-slim[tavily]"
```
  </Step>

  <Step title="Set Up API Keys" icon="key">
    Get your [Tavily API key](https://app.tavily.com/home) and set it as an environment variable in your terminal or add it to your environment configuration file:

```bash
export TAVILY_API_KEY=your_tavily_api_key
```
  </Step>

  <Step title="Initialize Pydantic AI Agent with Tavily Tools" icon="settings">
```python
import os
from pydantic_ai.agent import Agent
from pydantic_ai.common_tools.tavily import tavily_search_tool

# Get API key from environment
api_key = os.getenv('TAVILY_API_KEY')
assert api_key is not None

# Initialize the agent with Tavily tools
agent = Agent(
    'openai:o3-mini',
    tools=[tavily_search_tool(api_key)],
    system_prompt='Search Tavily for the given query and return the results.'
)
```
  </Step>

  <Step title="Example Use Cases" icon="code">
```python
# Example: Basic search for news
result = agent.run_sync('Tell me the top news in the GenAI world, give me links.')
print(result.output)
```

  </Step>
