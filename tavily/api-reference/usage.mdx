<ParamField
  name="GET /usage"
  location="path"
  type="endpoint"
  description="Get Tavily API key and account usage details."
/>

<Callout kind="info">
This endpoint returns usage information for the API key used in the
<code>Authorization</code> header, including key-level and account-level usage and limits.
</Callout>

## HTTP request

```http
GET https://api.tavily.com/usage
```

## Authentication

<ParamField
  name="Authorization"
  location="header"
  type="string"
  required
  description="Bearer authentication header in the form `Bearer <token>`, where `<token>` is your Tavily API key (for example, `Bearer tvly-YOUR_API_KEY`)."
/>

```http
Authorization: Bearer tvly-YOUR_API_KEY
```

## Query parameters

This endpoint does not accept any query parameters.

## Response

### 200 OK

Usage details for the current API key and account.

```json
{
  "key": {
    "usage": 150,
    "limit": 1000
  },
  "account": {
    "current_plan": "Bootstrap",
    "plan_usage": 500,
    "plan_limit": 15000,
    "paygo_usage": 25,
    "paygo_limit": 100
  }
}
```

<ParamField
  name="key.usage"
  location="body"
  type="integer"
  description="Current usage count for the API key."
  example="150"
/>

<ParamField
  name="key.limit"
  location="body"
  type="integer"
  description="Usage limit for the API key. Returns `null` if unlimited (2147483647)."
  example="1000"
/>

<ParamField
  name="account.current_plan"
  location="body"
  type="string"
  description="The current subscription plan name."
  example="Bootstrap"
/>

<ParamField
  name="account.plan_usage"
  location="body"
  type="integer"
  description="Current usage count for the plan."
  example="500"
/>

<ParamField
  name="account.plan_limit"
  location="body"
  type="integer"
  description="Usage limit for the current plan."
  example="15000"
/>

<ParamField
  name="account.paygo_usage"
  location="body"
  type="integer"
  description="Current pay-as-you-go usage count."
  example="25"
/>

<ParamField
  name="account.paygo_limit"
  location="body"
  type="integer"
  description="Pay-as-you-go usage limit."
  example="100"
/>

### 401 Unauthorized

Returned when the API key is missing or invalid.

```json
{
  "detail": {
    "error": "Unauthorized: missing or invalid API key."
  }
}
```
