## Get started in 3 steps

This quickstart shows how to create an account, install the SDK, and run your first search request. No prior API experience is required.

<Steps>

<Step title="1. Get your free API key" icon="key">

Visit the [Tavily Platform](https://app.tavily.com) to sign up or log in. After you sign in, copy your API key from the dashboard.

<Card icon="key" href="https://app.tavily.com" title="Sign up" horizontal>
No credit card is required to start.
</Card>

</Step>

<Step title="2. Install Tavily" icon="download">

Install the SDK for your preferred language from your terminal or command prompt.

<CodeGroup tabs="Python,JavaScript">

```bash
pip install tavily-python
```

```bash
npm i @tavily/core
```

</CodeGroup>

If you use cURL, no installation is required.

</Step>

<Step title="3. Run your first search" icon="search">

Replace `tvly-YOUR_API_KEY` with your API key, then run the example below. It searches for "Who is Leo Messi?" and prints the response.

<CodeGroup tabs="Python,JavaScript,cURL">

```python
from tavily import TavilyClient

tavily_client = TavilyClient(api_key="tvly-YOUR_API_KEY")
response = tavily_client.search("Who is Leo Messi?")

print(response)
```

```javascript
const { tavily } = require("@tavily/core");

const tvly = tavily({ apiKey: "tvly-YOUR_API_KEY" });
const response = await tvly.search("Who is Leo Messi?");

console.log(response);
```

```bash
curl -X POST https://api.tavily.com/search \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer tvly-YOUR_API_KEY" \
  -d '{"query": "Who is Leo Messi?"}'
```

</CodeGroup>

You should see search results immediately.

</Step>

</Steps>
